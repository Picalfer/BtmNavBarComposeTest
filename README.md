Тестовое приложение для изучения кастомной навигации с Bottom Navigation Bar в Jetpack Compose.

## О проекте

Приложение демонстрирует создание кастомного bottom navigation bar с иконками и цветовой схемой. Включает три экрана с навигацией между ними.

**Основной функционал:**
- Кастомный Navigation Bar с иконками Material Icons
- Три экрана: Home, Episodes, Search
- Цветовая схема с оранжевым акцентом
- Правильная навигация с сохранением состояния

## Технологии

* Kotlin
* Jetpack Compose
* Compose Navigation
* Material 3
* Material Icons
